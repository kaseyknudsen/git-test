How to install VS Code's Live Server Extension

Install | Use | Live Server | Video | Google

Installation Instructions

  1. In Visual Studio Code, click the Extensions icon in the left column toolbar
  2. In the search bar that appears, type Live Server.
  3. Select the Live Server extension. It should be the top choice.
  4. In the page for Live Server that appears, click the green install button.

Usage Instructions

  • Server must be open to a folder to use Live Server.
  • In VS code's explorer, right click (windows) or control+click (masOS) on an HTML file. From the context menu that appears, select Open with Live Server.

How to Know if You Have Installed Live Server Correctly

  1. Right Click the file name
  2. "Open with live server" will appear
  3. The browser will open with your html page
